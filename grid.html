1. HTML-dokumentin perusrakenne

Jokainen HTML-sivu alkaa samasta rungosta. Selain tarvitsee nämä tagit, jotta se osaa tulkita sivun oikein.

Koko sivun rakenne näyttää tältä (käytä &lt; ja &gt; kun haluat näyttää tagit selaimessa):

<!doctype html>
<html lang="fi">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Otsikko</title>
    <!-- CSS-linkit ja Bootstrap tulevat tähän -->
  </head>
  <body>
    <!-- Kaikki näkyvä sisältö tähän -->
  </body>
</html>
Esimerkki renderöitynä:

Tämä on <body>-osion sisältöä

Selain näyttää vain <body>-osion.

Huom! <head>-osion sisältö ei näy käyttäjälle.
Bootstrap-vinkki: Käytä aina <meta name="viewport"> -tagia, jotta Bootstrapin responsiivinen grid toimii oikein kaikilla näytöillä.

2. Tärkeimmät tagit esittelyssä

Tässä kahdeksan keskeistä tagia, joita tarvitaan lähes jokaisessa HTML-sivussa. Jokainen esimerkki sisältää selityksen, koodin ja live-demoversion.

<html>

Koko dokumentin juuri. Kaikki muu sisältö tulee tämän sisään.

<html lang="fi"> ... </html>
Koko sivun juuri
<head>

Metatiedot, otsikko, CSS-linkit ja Bootstrap-CDN. Ei näy käyttäjälle.

<head>
  <title>Sivun nimi</title>
</head>
Ei näkyvää sisältöä
<body>

Näkyvä sisältö: tekstit, kuvat, linkit jne.

<body>
  <h1>Hei maailma!</h1>
</body>

Hei maailma!

<h1>

Pääotsikko (suurin otsikkotaso). Käytä vain kerran sivulla.

<h1>Pääotsikko</h1>

Pääotsikko

<p>

Kappale tekstiä. Automaattisesti riviväli.

<p>Tämä on kappale.</p>

Tämä on kappale tekstiä Bootstrap-sivulla.

<a>

Hyperlinkki toiseen sivuun tai paikkaan.

<a href="https://getbootstrap.com/">Linkki</a>
Avaa Bootstrapin kotisivu
<img>

Kuva. Pakolliset attribuutit: src ja alt.

<img src="kuva.jpg" alt="Kuvaus" class="img-fluid">
<br>

Rivinvaihto (ei sulkevaa tagia).

3. Sivun muotoilu CSS:llä ja valmiilla kirjastoilla

CSS-muotoilusäännöt

Voit muokata elementtien ulkoasua kolmella tavalla:

  • Inline: <p style="color: red; font-size: 1.2rem;">
  • Internal: <style> -tagi <head>-osiossa
  • External: erillinen styles.css-tiedosto, joka linkitetään <link rel="stylesheet">

Esimerkki CSS-säännöstä:

p {
  color: #212529;
  line-height: 1.6;
}
